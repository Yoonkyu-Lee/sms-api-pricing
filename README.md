# SMS API Pricing — compare 9 CPaaS providers across 90+ countries

**Live site: https://yoonkyu-lee.github.io/sms-api-pricing/**

Normalized public **outbound SMS API rates** from Twilio, Telnyx, Plivo, Sinch, Vonage, Bandwidth, SignalWire, AWS SNS and MessageBird — compared across 90+ countries, with a free monthly cost calculator.

- Prices are **facts** collected from each provider's public pricing pages (source link + collection date on every row).
- Per-country comparison, per-provider pages, head-to-head (X vs Y), and a cost calculator.
- Honest by design — see [Methodology & limits](https://yoonkyu-lee.github.io/sms-api-pricing/methodology.html). No affiliation with any provider.

Built from a data pipeline + static-site generator; rebuilt from live sources.
